FiveM Breathalyzer script designed for the Bulgarian Police. It simulates a realistic breath alcohol test, allowing officers to check drivers for alcohol levels in an immersive roleplay environment. The script is lightweight, easy to use, and intended to enhance police roleplay scenarios on FiveM servers.
